# Team Task Manager 🚀

A full-stack Django web application for managing team projects and tasks with role-based access control.

## 📌 Features

- User Authentication (Signup/Login/Logout)
- Role-Based Access (Admin & Member)
- Project Management
- Task Creation & Assignment
- Task Status Tracking
- Dashboard with:
  - Total Tasks
  - Completed Tasks
  - Overdue Tasks
- Admin Panel
- Responsive Bootstrap UI

---

## 🛠️ Tech Stack

### Backend
- Python
- Django
- Django ORM

### Frontend
- HTML
- CSS
- Bootstrap

### Database
- SQLite

---

## 👥 User Roles

### Admin
- Create projects
- Create and assign tasks
- Manage users

### Member
- View assigned tasks
- Update task status
